# 输氢管道投产混气规律模拟报告框架

## 1. 问题定义

- 12 km 输氢管道投产置换；入口侧 8% 管长为 N2 隔离段，其余为空气。
- t=0 后入口通入纯 H2，分析 H2/N2/Air 三组分摩尔分数分布、混气段、安全风险段和有效 N2 隔离段。

## 2. 模型与变量

- 主变量为摩尔分数/体积分数 x_i，不使用质量分数作为输出或安全判据。
- 控制方程采用弱可压一维多组分对流-弥散模型。
- O2 摩尔分数为 x_O2 = 0.21 x_Air，惰性组分为 x_inert = x_N2 + 0.79 x_Air。
- 说明一维对流-弥散是成熟模型，作品创新在输氢投产场景应用、验证体系和工程规律提炼。

## 3. 数值方法

- 一维 cell-centered 有限体积网格。
- 对流项采用 van Leer TVD 重构，扩散项采用中心差分，时间推进采用显式 Euler。
- 每一步执行非负修正和三组分归一化，保证 x_H2 + x_N2 + x_Air = 1。

## 4. 安全判据

- 保守可燃判据：0.04 <= x_H2 <= 0.75 且 x_O2 >= 0.05。
- Fr < 1 为强分层风险，1 <= Fr < 3 为中等分层风险，Fr >= 3 为较低分层风险。
- 一维模型不能捕捉水平管内 H2 浮升分层，只用于轴向置换和混气段演化评价。

## 5. 验证

- erfc 单界面解析解验证。
- 组分和验证。
- dx = 20 m、10 m、5 m 网格无关性。
- 数值弥散上界 D_num = u dx (1-CFL)/2 与工程弥散 K 对比。
- beta = 0.2、0.5、0.8 敏感性；默认 beta=0.5 作为保守工程闭式，不作为唯一标定结论。
- Austin-Palfrey 仅作为液体顺序输送界面混合量级旁证。

## 6. Task1 参数扫描

- 管径 D = 0.7、1.0、1.2、1.4 m。
- 速度 u = 5、7、9 m/s。
- 出口背压 p_back = 0.02、0.05、0.10 MPa 绝压。
- 推荐速度基于混气段最大长度、可燃风险时间积分和摩阻压降综合评分。
- 在默认 `K = beta u D` 模型下，同管径的混气段对速度和背压不敏感；推荐速度应基于 Fr 分层风险、置换工期和压降约束，而不是声称高速能显著缩短混气段。
- 报告中同时列出突破时刻指标，避免最终纯 H2 充满后 `mixed_length_final=0`、`effective_n2_min=0` 这类天然归零指标误导读者。

## 7. Task2 中断工况

- D = 1.2 m，u = 7 m/s，p_back = 0.10 MPa。
- H2/N2 前缘到达 30%L、60%L、80%L 时停输 300 s。
- 结论模板：一维轴向模型中，停输 5 min 后轴向分子扩散导致的混气段增长较小；但 Fr -> 0 表明真实水平管内浮升分层风险急剧增强。中断越晚，氮气隔离段被湍流弥散侵蚀越明显，有效隔离长度越小，恢复置换安全裕度越低。
- 补充分子扩散 1%-99% 宽度约 1 m 与重力流侵入 O(几百米) 的对比，明确停输风险主要来自分层/重力流而非轴向分子扩散。

## 8. 技术创新性

1. 反直觉规律：解析推导 `sigma^2 = 2 beta D L` 并由全扫描证实混气段由管径主导，与速度、背压对摩尔分数剖面解耦。
2. 可信度工程：解析解、组分和、库存-边界通量守恒、网格无关性、数值弥散上界和 beta 敏感性联动校验。
3. 多机制分层风险：流动阶段用密度 Froude 数筛查，停输阶段用分子扩散与重力流量级共同解释。
4. 基于解耦结论的推荐决策：在 `dp/p_back < 0.10` 硬约束内选择 Fr 最大的速度，避免用失效的混气段指标做推荐。
