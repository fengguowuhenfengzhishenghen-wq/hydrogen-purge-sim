# OpenFOAM local cases

This folder contains OpenFOAM engineering cases generated from the verified 1D
H2/N2/Air purge model.

## DN1200_u7_p010_stop60_scalarTransport

Generated by:

```bash
python tools/build_openfoam_scalar_case.py --case-id DN1200_u7_p010_stop60
```

Purpose:

- Validate an OpenFOAM project structure for the local shutdown window.
- Mesh a 1200 m x-z local slab with pipe-diameter height 1.2 m.
- Initialize scalar field `T` from the 1D H2 mole-fraction profile.
- Run `scalarTransportFoam` with shutdown velocity `U=0` and molecular diffusivity.
- Export VTK files for ParaView.

Run in WSL:

```bash
cd /mnt/c/Users/陈雨婷/Desktop/ppt/hydrogen_purge_sim/openfoam_cases/DN1200_u7_p010_stop60_scalarTransport
WM_PROJECT_DIR=/usr/share/openfoam FOAM_ETC=/usr/share/openfoam/etc blockMesh
WM_PROJECT_DIR=/usr/share/openfoam FOAM_ETC=/usr/share/openfoam/etc checkMesh
WM_PROJECT_DIR=/usr/share/openfoam FOAM_ETC=/usr/share/openfoam/etc scalarTransportFoam
WM_PROJECT_DIR=/usr/share/openfoam FOAM_ETC=/usr/share/openfoam/etc foamToVTK
```

Verified status:

- `blockMesh`: passed.
- `checkMesh`: Mesh OK.
- `scalarTransportFoam`: ran to 300 s.
- `foamToVTK`: exported VTK time series.

Important limitation:

This is not a full buoyant multi-species H2/N2/Air Navier-Stokes CFD model. It is
a passive scalar OpenFOAM scaffold for the shutdown local window. The separate
Python reduced x-z stratification model remains the current source for the
reported cross-section segregation metrics.

```bash
  --length-m 48 --nx 32 --ny 10 --nz 10 \
  --end-time 1 --dt 0.2
```

Run in WSL/OpenFOAM:

